# NEOVIM 整体化配置
基于 [nvim-basic-ide](https://github.com/LunarVim/nvim-basic-ide) 

## Install the config

确保你已经移除(备份)自己的nvim文件, 包括缓存文件

```sh 
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.cache/nvim
```

```sh
git clone git@github.com:yl4869/my_nvim_config.git ~/.config/nvim
```

运行 `nvim` 并等待插件安装


## 检查health

打开 `nvim` 并进行以下操作

```
:checkhealth
```

下载相关需要的依赖, 如`ripgrep` 等。
